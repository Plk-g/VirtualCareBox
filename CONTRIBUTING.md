# Contributing

Thanks for wanting to improve Virtual Care Box.

## Ways to contribute

- Fix typos / improve copy
- Improve accessibility (keyboard support, contrast, screen reader behavior)
- Reduce image sizes / improve performance
- Add small features (animations, more items, themes), keeping it lightweight

## Local setup

- Install Node (optional, for formatting)
- Run locally:

```bash
npx serve .
```

## Style

- Keep it simple: plain HTML/CSS/JS
- Prefer shared styles in `assets/styles/site.css`
- Keep JS in `assets/scripts/` and avoid heavy dependencies
- Format with:

```bash
npm install
npm run format
```

## Submitting changes

- Create a branch
- Keep PRs small and focused
- Include a short description + screenshots if the UI changed
